# Bước 3- Xoá kênh

Sau khi đã tạo kênh và khởi chạy, khi bạn bạn cần dừng kênh và đóng kênh vui lòng thao tác sau:

**Để xoá kênh**

1. Tại mục **danh sách kênh**, chọn kênh cần xoá, Click vào icon **Xoá**
2. Hiển thị **Pop-up xác nhận**, chọn nút **Xác nhận**.
